# Greeting Card

Ứng dụng Android đơn giản được xây dựng bằng Kotlin và Jetpack Compose.

## Giới thiệu

Đây là project luyện tập Android đầu tiên sử dụng Jetpack Compose để tạo giao diện thiệp chúc mừng.

Ứng dụng hiển thị:
- Tin nhắn chúc mừng
- Tên người gửi
- Bố cục giao diện bằng Column
- Modifier để căn chỉnh giao diện

## Công nghệ sử dụng

- Kotlin
- Jetpack Compose
- Material 3
- Android Studio

## Kiến thức đã học

- `@Composable`
- `Text`
- `Column`
- `Modifier`
- `padding()`
- `align()`
- `fontSize`
- `Surface`
- `MaterialTheme`
